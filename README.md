# Chicken Production Sheet by shai

written using python to track and manage chicken production cost.

## Features
- Automatically calculate cost for parts
- GUI for easy data entry
- Supports saving and locking cost

## Installation
1. Clone the repository:
```bash
git clone https://github.com/xodez98-web/production-sheet.git
``` 

2. install dependecies

```bash
pip intall -r requirements.txt
```
3. Run the python script:
```bash
python ps_chicken_gui.py
```
## Usage
- run .exe
- enter chicken quantities and costs
- click "save" to save and lock the data
- follow prompts for any invalid input
